# Changelog

All notable changes to the "Competitive Coding Helper" extension will be documented in this file.

and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-07

### Added
- Random problem fetching by difficulty and topic
- Direct HackerRank problem import via URL
- Automated test case execution
- Smart test case generation based on problem patterns
- Browser-based submission with clipboard integration
- Complete problem statement extraction with UTF-8 support
- Support for C++, Python, Java, JavaScript, and Go
- Workspace organization with problem folders
- Sample and generated test case management
- Production-level error handling
- Comprehensive documentation
